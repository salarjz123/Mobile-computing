Sketches of the final app

Loading Page

This is the loading page of the application, it shows the name and logo of the application. This will be displayed while the application is loading to the main landing page.

Landing Page

This is the main landing page of the app, it will take the user automatically into the dictionary. The user can click on the left menu to change to thesaurus or select other options

Left Menu

When the user presses the button for the left menu on the screen then the options will be prompted.

Dictionary Search Page

The above screen indicates a user searching the word Goal in Dictionary, the app will also give suggestions based on what the user is searching for. The user can also use their voice to search the word if they click on the microphone button.

Thesaurus Search Page

Like the word search in the previous screen for dictionary, this is a word search in thesaurus.

Meaning View Page

Above is an example of a word search in dictionary, Goal in this case. The app displays the type of word, noun in this case, and the suitable meanings of it.

Synonym View Page

Above we can see that if the user selects the word Life in Thesaurus, then the app displays the type of word it is (noun in this case) and a few synonyms of it.

History View Page

If the user presses the History button on the main landing page, then they will be taken to this page where they can see their past word searches for dictionary and thesaurus. The user will also have the option to delete the history for both dictionary and thesaurus if they press the Delete history button.
